topic#5: continue



we have learned how to declare and integer type


but we haven't assigned a value to our variable yet. A variable is just like and empty room . let's put sombody there


in c#. there are two ways that we can use to assign values to variables


  • on thesame line when declaring it.

  • assigning it later in the code post declaration

    • variables in c# can change their values when going through various operations


    C# was initialy named C-Cool but it was dropped due to copyright issues


    let's see how the code looks like


    the first example illustrates the declaration and value assigning process on thesame line of code


    Sample Code

            
Using System;
namespace mynamespace{
  class Program{
    static void Main(string[], args){
      int A = 10;
      int B = -50;
      int C = 0;
    }
  }
}

    Output


    So what we got are 3 integers variable A, B, C, containing 3 different values 10, -50, 0 respectively



    the second example is asigning a value for a variable after it has been declared.


    Sample Code

            
Using System;
namespace   CodeSample{
  class Program {
    static void Main(string[] args){
      int A;
      int B;
      A = 20;
      B = 12;
    }
  }
}

    Output


    in order to store decimals in c#, we need to choose the right variable. we will use Double in This case


    Sample Code

    Using System;
namespace   CodeSample{
  class Program {
    static void Main(string[] args){
      double A;
    }
  }
}

    Output


    double is the type of variable used for decimals
