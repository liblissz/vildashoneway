CSS Overview

What is CSS?

CSS stands for Cascading Style Sheets. It's a styling language used to control the layout and appearance of web pages written in HTML or XML.

Features of CSS

  • Selectors: Used to target specific HTML elements.
  • Properties: Define the style or layout of an element.
  • Values: Assigned to properties to specify the style or layout.
  • Cascading Styles: Styles can be applied from multiple sources, with later styles overriding earlier ones.

Types of CSS

  • Internal CSS: Written directly in an HTML document.
  • External CSS: Written in a separate file, linked to an HTML document.
  • Inline CSS: Written directly in an HTML element.

CSS Syntax

selector { property: value; }

Example: h1 { color: blue; }

CSS Selectors

  • Element Selectors: h1, p, img
  • Class Selectors: .class-name
  • ID Selectors: #id-name
  • Attribute Selectors: [attribute="value"]

CSS Properties

  • Color Properties: color, background-color
  • Font Properties: font-family, font-size
  • Layout Properties: width, height, margin, padding
  • Border Properties: border-style, border-width, border-color

CSS Units

  • Length Units: px, em, cm, mm, in, pt, pc
  • Percentage Units: %
  • Color Units: rgb(), rgba(), hex

CSS Box Model

  • Content Area: The area where the content is displayed.
  • Padding: The space between the content and the border.
  • Border: The visible outline of the box.
  • Margin: The space between the box and other elements.

CSS Positioning

  • Static Positioning: The default positioning scheme.
  • Relative Positioning: Positions an element relative to its normal position.
  • Absolute Positioning: Positions an element relative to its nearest positioned ancestor.
  • Fixed Positioning: Positions an element relative to the viewport.
And here are the remaining sections:

CSS Display

  • Block Display: Occupies the full width of its parent.
  • Inline Display: Occupies only the space needed for its content.
  • Inline-Block Display: Combines features of block and inline display.

CSS Flexbox

  • Flex Container: The element that contains the flex items.
  • Flex Items: The elements that are direct children of the flex container.
  • Flex Direction: The direction in which the flex items are laid out.

CSS Grid

  • Grid Container: The element that contains the grid items.
  • Grid Items: The elements that are direct children of the grid container.
  • Grid Tracks: The horizontal and vertical lines that make up the grid.
