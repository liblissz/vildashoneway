LEARN HTML

HTML Elements

Headings

Headings are defined with the <h1> to <h6> tags.

<h1>Heading 1</h1> <h2>Heading 2</h2> <h3>Heading 3</h3> <h4>Heading 4</h4> <h5>Heading 5</h5> <h6>Heading 6</h6>

Paragraphs

Paragraphs are defined with the <p> tag.

<p>This is a paragraph of text.</p>

Links

Links are defined with the <a> tag.

<a href="(link unavailable)">Visit Example</a>

Images

Images are defined with the <img> tag.

<img src="image.jpg" alt="An image">

Lists

Lists are defined with the <ul>, <ol>, and <li> tags.

<ul> <li>Item 1</li> <li>Item 2</li> <li>Item 3</li> </ul>

Tables

Tables are defined with the <table>, <tr>, and <td> tags.

<table> <tr> <td>Cell 1</td> <td>Cell 2</td> </tr> <tr> <td>Cell 3</td> <td>Cell 4</td> </tr> </table>

Forms

Forms are defined with the <form> tag.

<form> <label>Name:</label> <input type="text" name="name"> <br> <label>Email:</label> <input type="email" name="email"> <br> <input type="submit" value="Submit"> </form>

Semantic Elements

Semantic elements provide meaning to the structure of the web page.

<header>Header</header> <nav>Navigation</nav> <main>Main content</main> <section>Section</section> <article>Article</article> <aside>Aside</aside
