I.Class Basic Syntax

In practice , we often need to create many objects of the same kind , like users , or goods or whatever
As we already know from the chapter Constructor,operator"new", new function can help with that.
But in the mordern JavaScript,there is a more advanced "class" Construct, tha introduces great new features which are useful for object-oriented programming.
The "class" Syntax The basic syntax is : class Myclass { // class methods constructor() {....} method1(){...} method2(){...} method3(){...} .... } Then use Myclass() to creat a new object with all the listed methods.
The constructor() method is called automatically by new, so we can initialize the object there.
for Example <html> <script> class user { constructor(name) { this.name = name; } } </script> </html> // Usage: let user = User("John"); user.sayHi(); What is a class??
so what exactly is a class? That's not an entirely new languge-level entity,as one might think Let's unveil any magic and see what a class really is.That'll helpm in understanding many complex aspects. In javascript, a class is a kind of function.
Here take a look: <html> <script> class user { constructor(name) { this.name = name ; } sayHi(){ alert(this.name);} } //proof : User is a function alert(tyoeofUser); // function </script> </html> What class User{...} construct really does is:
- Creates a function named User, that becomes the result of the class declaration. The function is taken from the constructor method (assumed empty if we don't write such method)
- Stores class mehods, such as sayHi, in User.prototype.
After new user object is created, when we call its method, it's taken from the prototype,just as described in the chapter F.prototype. So the object has access to class methods.
