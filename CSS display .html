CSS Display Property

The CSS display property is used to define the type of box used to display an element.

Display Values

The following are the possible values for the display property:

  • block: The element is displayed as a block element, taking up the full width of its parent.
  • inline: The element is displayed as an inline element, taking up only the space needed for its content.
  • inline-block: The element is displayed as an inline-level block element.
  • none: The element is not displayed.
  • flex: The element is displayed as a flexible box.
  • grid: The element is displayed as a grid container.
