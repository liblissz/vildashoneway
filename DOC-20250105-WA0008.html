TOPIC1#:INTRODUCTION TO PYTHON

Python is a high-level programming language that is widely used for a variety of applications. It is also interpreted,interactive and object-oriented scripting language.Python is designed to be highly readable.It uses English Keywords frequently where as other languages uses punctuation, and it haas fewer syntactical constructions than other languages.It is known for its simplicity and readability, making it an excellent choice for beginners and professionals alike.

Key Festures of Python

1. Easy to Learn and Use:

Python's syntax is simple and close to natural language,which makes it beginner-friendly It has few keywords,simple structure and a clearly defined syntax.This allow the student to pick the language quickly.

print("hello,world")

2. interpreted language:

Python executes code line by line, which simplifies debugging. python is processed at runtime by the interpreter.You do notm need to compile your program before executing it. TThis is similar to PERL and PHP.

3.A broad standard library:

Pyhton's bulk of the library is very portable and cross-platform compartible on UNIX ,WINDOWS,MACINTOSH.

4. Portable:

Python can run on a wide variety of hardware platforms and has the same interface on all platforms.

5. Extendable:

You can add low-level modules to the python interpreter.These modules enables programmers to add to or customize their tools to be more efficient.

Brief History of Python

Python was developed by Guido van Rossum in the late 80s and early 90s at the National Research Institute for Mathematics and Computer Science in the NetherLands.

What can You do with Python??

- WEb development: Build websites using frameworks like Django and Flask.

-Data Science and AI: Analyze data and build machine learning models with libraries like pandas and scikit-learn.

-Game Development: Create games using repetitive task.

-Desktop Applications: Build GUIs with Tkinter or PyQt.

TOPIC3# VARIABLE TYPES:

                                 

                                  
Varirables

                                  
variables are nothing but reserved memory locations to store values. This means that 
                                    when you create a variable you reserve some space in the memory. 

                                    Based on the data type of a variable,the interpreter allocates memory and decides what
                                    can be stored in the reserved memory. Therefore, by assigning different data types to variables
                                    you can store integers, decimals or characters in these variables.
                                  

                                  
Assigning Values to variables

                                  
Python variables do not need explicit declaration to reserve memory space. The 
                                    declaration happens automatically when you assign a value to a variable.The equal
                                    Sign(=) is used to assig values to variables 

                                    The operand to the left of the = operator is the name of the variable  and the operand
                                    to the right of the = operator is the value stored in the variables.For example-
                                  

                                  
                                    #!/user/bin/python  
                                    counter = 100        # An  interger assignment
                                    mile = 1000.0        # A float point
                                    name = "John"        # A string



                                    print counter
                                    print miles
                                    print name
                                  
                    Here, 100, 1000.0 and "John" are the values assigned to couter,miles,and name variables, repectively
                    This produces the following results- 
                    
                      100
                      1000.0
                      John
