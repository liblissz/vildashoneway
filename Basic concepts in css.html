LEARN CSS CHAPTER1

1. Syntax

CSS syntax consists of a selector, property, and value. The selector is used to target an HTML element, the property is the style attribute you want to change, and the value is the new value you want to assign to the property.

h1 {
color: blue;
font-size: 36px;
}

2. CSS Comments

CSS comments start with /* and end with _/. They are used to add notes to your CSS code.

/_ This is a CSS comment _/
h1 {
color: blue; /_ This is another CSS comment */
font-size: 36px;
}

CSS comments are ignored by the browser and are used to explain the code to other developers or to yourself.

3. Inline CSS

Inline CSS is used to style a single element. It is added directly to the HTML element using the style attribute.

<h1 style="color: blue; font-size: 36px;">This heading is blue and 36px</h1>

Inline CSS is useful for making small changes to a single element, but it can make the code harder to read and maintain.

4. Internal CSS

Internal CSS is used to style a single HTML document. It is added to the HTML document using the style element.

  
  <head>
<style>
h1 {
color: blue;
font-size: 36px;
}
</style>
</head>

Internal CSS is useful for making changes to a single HTML document, but it can make the code harder to read and maintain.

5. External CSS

External CSS is used to style multiple HTML documents. It is added to a separate CSS file and linked to the HTML document using the link element.

<link rel="stylesheet" type="text/css" href="styles.css">

External CSS is useful for making changes to multiple HTML documents at once, and it can make the code easier to read and maintain.

6. CSS Selectors

CSS selectors are used to target HTML elements. There are several types of selectors, including:

  • Element selectors (e.g. h1, p, img)
  • Class selectors (e.g. .class-name)
  • ID selectors (e.g. #id-name)
  • Attribute selectors (e.g. [attribute="value"])
h1 {
color: blue;
font-size: 36px;
}

CSS selectors are used to apply styles to specific HTML elements.
