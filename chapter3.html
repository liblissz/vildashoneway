HTML Features Tutorial

Welcome to this tutorial on HTML features!

Basic HTML Structure

The basic structure of an HTML document consists of the following elements:

  • <!DOCTYPE html>: The document type declaration
  • <html>: The root element of the document
  • <head>: The head section of the document
  • <title>: The title of the document
  • <body>: The body section of the document

HTML Styles

HTML styles are used to control the layout and appearance of HTML elements.

There are three types of HTML styles:

  • Inline styles: These styles are applied directly to an HTML element using the style attribute.
  • Internal styles: These styles are applied to an HTML document using the <style> element.
  • External styles: These styles are applied to an HTML document using an external stylesheet.

Responsive Design

Responsive design is the practice of designing websites that adapt to different screen sizes and devices.

Media queries are used to apply different styles based on different screen sizes.

Centering Text

Centering text can be achieved using the text-align property.

Text Formatting

Text formatting can be achieved using various HTML elements and attributes.

Bold text can be achieved using the <b> element or the font-weight property.

Italic text can be achieved using the <i> element or the font-style property.
