Conditional Statements in Java (IF-ELSE, IF-ELSE IF)

Conditional statements in Java are fundamental for making decisions and controlling the flow of a program based on certain conditions. These statements allow developers to execute different blocks of code depending on whether a condition evaluates to true or false. Among the most commonly used conditional statements are IF-ELSE and IF-ELSE IF, which provide flexibility for implementing complex decision-making logic.

The IF-ELSE Statement

The IF-ELSE statement is a two-way decision-making construct that executes one block of code if a specified condition is true and another block if the condition is false. It is ideal for situations where a binary decision is needed, such as validating user input or determining whether a number is positive or negative. The syntax for IF-ELSE is straightforward, making it easy to use in simple and complex scenarios.

The IF-ELSE IF Statement

The IF-ELSE IF construct, on the other hand, extends the decision-making capabilities by allowing multiple conditions to be evaluated in sequence. This is particularly useful when there are more than two possible outcomes. The program evaluates each condition in the order they appear, and the first condition that evaluates to true determines which block of code is executed. If none of the conditions are true, an optional ELSE block can be used as a default case.

Example 1: Simple IF-ELSE Statement

    
        public class IfElseExample {
            public static void main(String[] args) {
                int number = 10;
        
                if (number > 0) {
                    System.out.println("The number is positive.");
                } else {
                    System.out.println("The number is not positive.");
                }
            }
        }

Example 2: Using IF-ELSE IF for Multiple Conditions

    
        public class IfElseIfExample {
            public static void main(String[] args) {
                int score = 85;
        
                if (score >= 90) {
                    System.out.println("Grade: A");
                } else if (score >= 80) {
                    System.out.println("Grade: B");
                } else if (score >= 70) {
                    System.out.println("Grade: C");
                } else {
                    System.out.println("Grade: F");
                }
            }
        }

Example 3: Nested IF-ELSE Statement

    
        public class NestedIfElseExample {
            public static void main(String[] args) {
                int age = 20;
                boolean hasPermission = true;
        
                if (age >= 18) {
                    if (hasPermission) {
                        System.out.println("Access granted.");
                    } else {
                        System.out.println("Permission required.");
                    }
                } else {
                    System.out.println("Access denied. Age restriction applies.");
                }
            }
        }
