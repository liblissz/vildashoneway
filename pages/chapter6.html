Loops (WHILE, FOR, DO-WHILE) in Java

In Java, loops are used to execute a block of code repeatedly based on certain conditions. They are essential for tasks that require iteration, such as processing elements in a collection, handling repetitive tasks, and performing calculations. There are three primary types of loops in Java: the while loop, for loop, and do-while loop, each of which has a unique way of handling iterations and conditions.

The WHILE loop

The while loop repeatedly executes a block of code as long as the specified condition evaluates to true. If the condition is false initially, the code inside the loop is never executed. This type of loop is useful when the number of iterations is not known in advance, but the condition to continue looping is clear. An example of using a while loop might be when reading data until no more data is available or counting down from a certain number.

The FOR loop

The for loop is typically used when the number of iterations is known beforehand. It provides a more concise and flexible syntax for initialization, condition checking, and incrementing. This loop is ideal for iterating over a range of values, such as processing elements in an array or performing a task a fixed number of times. Unlike the while loop, the for loop combines all three parts (initialization, condition, and update) in a single statement, making it very efficient for cases with a defined limit.

The DO-WHILE loop

The do-while loop is similar to the while loop, but with one key difference: it guarantees that the loop body is executed at least once, regardless of whether the condition is true or false. This loop is particularly useful when you want the code inside the loop to run at least one time before checking the condition. After the first iteration, it behaves like a standard while loop, checking the condition at the end of each iteration.

Example 1: WHILE Loop

    
        public class WhileLoopExample {
            public static void main(String[] args) {
                int count = 0;
                while (count < 5) {
                    System.out.println("Count: " + count);
                    count++;
                }
            }
        }

        /* After excution of the code 

        Count: 0
        Count: 1
        Count: 2
        Count: 3
        Count: 4
 */

Example 2: FOR Loop Example:

    
        public class ForLoopExample {
            public static void main(String[] args) {
                for (int i = 0; i < 5; i++) {
                    System.out.println("i: " + i);
                }
            }
        }
        /*After execution of code
            i: 0
            i: 1
            i: 2
            i: 3
            i: 4
        */

Example 3: DO-WHILE Loop

    
        public class DoWhileLoopExample {
            public static void main(String[] args) {
                int count = 0;
                do {
                    System.out.println("Count: " + count);
                    count++;
                } while (count < 5);
            }
        }
        /* After execution of code
            Count: 0
            Count: 1
            Count: 2
            Count: 3
            Count: 4
*/

Example 3: FOR Loop to Iterate Over an Array:

    
        public class ArrayForLoopExample {
            public static void main(String[] args) {
                int[] numbers = {10, 20, 30, 40, 50};
                for (int i = 0; i < numbers.length; i++) {
                    System.out.println("Number: " + numbers[i]);
                }
            }
        }
        
        /* After execution of code
        Number: 10
        Number: 20
        Number: 30
        Number: 40
        Number: 50        
*/
