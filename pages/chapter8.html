Introduction to Methods and Objects

Methods and objects are foundational concepts in Java's object-oriented programming (OOP) paradigm. A method is a block of code designed to perform a specific task, and it can be executed whenever it is called. Methods improve code reusability and modularity, allowing developers to write clean and maintainable programs. An object, on the other hand, is an instance of a class that encapsulates data (fields) and behavior (methods). By using methods and objects together, developers can simulate real-world interactions, making Java programs more intuitive and scalable.

Defining and Using Methods

Methods in Java are defined within a class and can have different types based on their purpose. There are static methods, which belong to the class and can be called without creating an object, and instance methods, which require an object for invocation. Methods can take parameters and return values, enabling them to perform calculations or transformations. The public, private, or protected access modifiers control the visibility of a method, ensuring encapsulation and secure data handling.

Object-Oriented Programming with Methods and Objects

Objects allow the bundling of related fields and methods into a single unit. By creating multiple objects from a class, each object can have its unique state while sharing the same behavior. This promotes code reuse and abstraction. Java provides powerful features like method overloading and overriding, allowing developers to define multiple methods with the same name but different parameters or redefine inherited methods for specific behavior in subclasses.

Example 1: Defining and Calling a Method

    
        public class Greetings {
            // Method definition
            public void sayHello() {
                System.out.println("Hello, welcome to Java programming!");
            }
        
            public static void main(String[] args) {
                // Creating an object to call the method
                Greetings greet = new Greetings();
                greet.sayHello();
            }
        }

Example 2: Method with Parameters and Return Value

    
        public class Calculator {
            // Method with parameters and return value
            public int add(int a, int b) {
                return a + b;
            }
        
            public static void main(String[] args) {
                Calculator calc = new Calculator();
                int result = calc.add(5, 10);
                System.out.println("The sum is: " + result);
            }
        }

Example 3: Object-Oriented Programming Example

    
        class Car {
            String model;
        
            // Constructor
            public Car(String model) {
                this.model = model;
            }
        
            // Method
            public void displayModel() {
                System.out.println("Car model: " + model);
            }
        }
        
        public class Main {
            public static void main(String[] args) {
                Car car1 = new Car("Tesla Model S");
                Car car2 = new Car("Ford Mustang");
        
                car1.displayModel();
                car2.displayModel();
            }
        }

Example 4: Static and Instance Methods

    
        public class MathUtils {
            // Static method
            public static int square(int number) {
                return number * number;
            }
        
            // Instance method
            public double squareRoot(double number) {
                return Math.sqrt(number);
            }
        
            public static void main(String[] args) {
                // Calling static method
                int squared = MathUtils.square(4);
                System.out.println("Squared value: " + squared);
        
                // Calling instance method
                MathUtils mathUtils = new MathUtils();
                double squareRoot = mathUtils.squareRoot(16.0);
                System.out.println("Square root: " + squareRoot);
            }
        }
